# Financial Analytics Dashboard (Power BI & SQL)

## Overview
This is my first GitHub project created to demonstrate core Financial Analyst skills.  
The project focuses on financial data analysis using SQL, Power BI, and AI-assisted insights.

## Tools & Technologies
- Power BI (Data Visualization & KPI Dashboards)
- SQL (Data Analysis Queries)
- Microsoft Excel (Data Preparation)
- AI Tools (Insight generation and business explanation)

## Project Structure
- data/ – Raw datasets used for analysis
- sql/ – SQL queries for financial analysis
- insights/ – AI-assisted insights and observations

## Key Objectives
- Analyze financial data to identify trends and patterns
- Build KPI-focused dashboards for decision-making
- Demonstrate structured analytical thinking
- Showcase a beginner-friendly end-to-end analytics workflow

## Notes
This project was built as a learning and portfolio exercise and reflects a practical,  
business-focused approach to financial analytics.
